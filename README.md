# AI-Based Crop Disease Detection

## 📖 Project Description
This project detects crop diseases using a Convolutional Neural Network (CNN) model. 
Users can upload a crop leaf image, and the system predicts the disease class.

## 🚀 Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- CNN (Deep Learning)

## 📂 Project Structure
- backend/ → Model training and prediction logic
- frontend/ → User interface (Streamlit app)
- dataset/ → Dataset information
- model/ → Trained CNN model (.h5 file)
- screenshots/ → Project output screenshots

## 🧠 How It Works
1. User uploads leaf image.
2. Image is preprocessed.
3. CNN model predicts disease class.
4. Result is displayed on screen.

## 🔮 Future Scope
- Real-time detection using live camera
- Mobile application deployment
- Drone-based crop monitoring
- Integration with farmer advisory systems

## 📊 Dataset
PlantVillage Dataset (Kaggle)
